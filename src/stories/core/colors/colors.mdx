# Usage

### React Component
```jsx
import { Colors } from "mili-ui";

<div style={{ color: Colors.BLUE3, background: Colors.BLACK }} />
```

### Sass
```css
@import "mili-ui/core/lib/scss/variables";

.rule {
    color: $red;
    background: $secondary-color;
}
```
